# Umbracryptum

Customs scripts used in our paper describing the cultivation of Candidatus 'Yngarchaeum umbracryptum'

# Operon_singleton_opposite_strand_counter.py

A python script for annotating and counting the number of gene clusters (called operons in script), singletons and nearby genes on the opposite strand

# Operon_singleton_associated_predictor.py

A python script for detecting gene clusters (called operons in script), singletons and nearby genes on the opposite strand and printing a genbank record of each for each genome. Also produces a gene function .csv file. Can be inputted into software for illustrating operon/gene cluster structures
